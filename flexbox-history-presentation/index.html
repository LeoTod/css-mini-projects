Where It All Began

The CSS Flex layout first saw daylight in July 2009, nearly a year after it was first proposed by the CSS Working Group. However, no prototype is without its flaws, and the notoriously slow XUL-based Flex layout was soon scrapped for something much better.

Growing Pains

In 2011, Tab Atkins Jr. (often credited for the original layout) decided it was time for a fresh start. The initial slow algorithm and lack of documentation weren't going to cut it. His main goal was to try and remove the dependency on all the crazy CSS hacks he had to master as a web developer designing layouts and implement better browser support.

Internet Domination

After 3 years of trials and tribulations, the 2012 version of Flexbox was put into W3C candidate recommendation. Since then, the Flexbox spec has remained relatively stable, and the resulting near-universal support has propelled our favorite CSS layout to become the goliath it is today.
